## 2026-03-07 - Focus Rings for Icon-Only Buttons in Dark Theme
**Learning:** In a dark-theme UI like web-oven, icon-only buttons (such as dismiss/close buttons) can easily lose their keyboard accessibility. The standard browser focus ring often blends into the dark `slate` background. Adding a high-contrast focus ring using Tailwind's `focus-visible:ring-2 focus-visible:ring-orange-500` ensures keyboard navigators can clearly see when these small interactive elements have focus, matching the global accent color. Additionally, without text, screen readers need an `aria-label` to announce the action.
**Action:** Always verify icon-only buttons have an `aria-label` and explicitly set `focus-visible` classes with the `orange-500` accent color to guarantee visibility against dark backgrounds.
